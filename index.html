Fuck me on GitHub–CSS ribbon

This is a pure CSS implementation of the famous Fuck me on GitHub ribbon.

Usage

To use this on your site, check the soure. You'll want the commented CSS and HTML tags. You can also just go right to the repo and grab the files.

Using with Bower

You can install the CSS file using Bower. 

bower install f-me-on-github
